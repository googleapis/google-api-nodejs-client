# dlp

> Discover and protect your sensitive data. A fully managed service designed to help you discover, classify, and protect your valuable data assets with ease.

## Support status
**Note**: Google provides multiple libraries for this service. This library is in
maintenance mode, and will continue to be made available for users who have
existing applications. If you're building a new application, or modernizing a
legacy application, please use [@google-cloud/dlp](https://www.npmjs.com/package/@google-cloud/dlp) instead.
The [@google-cloud/dlp](https://www.npmjs.com/package/@google-cloud/dlp) library is faster, easier to use, and better maintained.

To learn more, see [Client Libraries Explained](https://cloud.google.com/apis/docs/client-libraries-explained).

## Installation

```sh
$ npm install @googleapis/dlp
```
