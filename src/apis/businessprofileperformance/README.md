# businessprofileperformance

> The Business Profile Performance API allows merchants to fetch performance reports about their business profile on Google. Note - If you have a quota of 0 after enabling the API, please request &lt;https://developers.google.com/my-business/content/prereqs#request-access&gt; for GBP API access.

## Installation

```sh
$ npm install @googleapis/businessprofileperformance
```

## Usage
All documentation and usage information can be found on [GitHub](https://github.com/googleapis/google-api-nodejs-client).
Information on classes can be found in [Googleapis Documentation](https://googleapis.dev/nodejs/googleapis/latest/businessprofileperformance/classes/Businessprofileperformance.html).
